นโยบายความเป็นส่วนตัว

ครุจิตเป็นเว็บไซต์อิสระเพื่อการศึกษา ให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้ และจัดเก็บข้อมูลให้น้อยที่สุดเท่าที่จำเป็น นโยบายนี้อธิบายว่าเราเก็บข้อมูลอะไร เพื่ออะไร และผู้ใช้มีสิทธิอย่างไร ตามแนวทางพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA)

ปรับปรุงล่าสุด: 29 มิถุนายน 2569

1. ภาพรวม

ครุจิตเป็นเว็บไซต์แบบ static (เผยแพร่ผ่าน GitHub Pages) ไม่มีระบบสมาชิก ไม่มีการเข้าสู่ระบบ และไม่ได้เก็บรหัสผ่านหรือข้อมูลทางการเงินใด ๆ การเข้าชมเนื้อหาทั่วไปสามารถทำได้โดยไม่ต้องระบุตัวตน

2. ข้อมูลที่จัดเก็บและวิธีการจัดเก็บ

2.1 ข้อมูลการเข้าถึงเว็บไซต์ (ไม่ระบุตัวตนโดยตรง)

ผู้ให้บริการโฮสต์ (GitHub Pages) อาจบันทึกข้อมูลทางเทคนิคโดยอัตโนมัติ เช่น หมายเลข IP ชนิดเบราว์เซอร์ และเวลาที่เข้าถึง เพื่อความปลอดภัยและการให้บริการตามมาตรฐานทั่วไปของผู้ให้บริการ ครุจิตไม่ได้นำข้อมูลเหล่านี้มาวิเคราะห์เป็นรายบุคคล

2.2 ข้อมูลที่จัดเก็บในเบราว์เซอร์ของผู้ใช้ (localStorage)

หน้าคลังข้อสอบฝึกฝนจะบันทึก "คะแนนสูงสุด" ของผู้ใช้ไว้ใน localStorage ของเบราว์เซอร์ บนเครื่องของผู้ใช้เองเท่านั้น ข้อมูลนี้ไม่ถูกส่งกลับมายังเซิร์ฟเวอร์หรือบุคคลที่สาม และผู้ใช้สามารถลบได้เองโดยล้างข้อมูลเว็บไซต์ (clear site data) ในเบราว์เซอร์

2.3 ข้อมูลจากแบบฟอร์มแจ้งข้อมูล/เสนอแนะ

หากผู้ใช้เลือกกรอกแบบฟอร์มแจ้งความคลาดเคลื่อน/เสนอแนะ (ผ่าน Google Forms) ระบบจะจัดเก็บเฉพาะข้อมูลที่ผู้ใช้กรอกเอง เช่น รายละเอียดที่แจ้ง และอีเมล (เฉพาะกรณีที่ผู้ใช้ประสงค์ให้ติดต่อกลับ) ข้อมูลนี้จัดเก็บบนระบบของ Google และนำมาใช้เพื่อปรับปรุงเว็บไซต์และตอบกลับเท่านั้น เราขอความร่วมมือผู้ใช้หลีกเลี่ยงการส่งข้อมูลส่วนบุคคลที่ไม่จำเป็น เช่น เลขประจำตัวประชาชน ข้อมูลเด็ก หรือข้อมูลสุขภาพ

3. คุกกี้และการติดตาม

ปัจจุบันครุจิต ไม่ใช้คุกกี้เพื่อการตลาดหรือการโฆษณา และไม่มีระบบติดตามพฤติกรรม เช่น Google Analytics, โฆษณา หรือ tracking pixel ใด ๆ มีเพียงการใช้ localStorage เชิงเทคนิคตามข้อ 2.2 หากในอนาคตมีการเพิ่มเครื่องมือวิเคราะห์ จะปรับปรุงนโยบายนี้และแจ้งให้ทราบ

4. บริการของบุคคลที่สาม

เพื่อให้เว็บไซต์ทำงานได้ ครุจิตใช้บริการของบุคคลที่สามบางส่วน ซึ่งแต่ละบริการมีนโยบายความเป็นส่วนตัวของตนเอง

  • GitHub Pages — ผู้ให้บริการโฮสต์เว็บไซต์
  • Google Fonts — ใช้แสดงแบบอักษร ซึ่งอาจมีการเรียกข้อมูลและบันทึกหมายเลข IP เมื่อโหลดฟอนต์
  • Google Forms — ใช้รับแบบฟอร์มแจ้งข้อมูล/เสนอแนะ

5. วัตถุประสงค์ของการใช้ข้อมูล

ข้อมูลที่ได้รับใช้เพื่อปรับปรุงความถูกต้องและคุณภาพของเนื้อหา ตอบกลับข้อเสนอแนะ และดูแลความปลอดภัยของเว็บไซต์เท่านั้น ครุจิตไม่ขาย ไม่ให้เช่า และไม่แลกเปลี่ยนข้อมูลส่วนบุคคลเพื่อการตลาดกับบุคคลภายนอก

6. ระยะเวลาการจัดเก็บ

ข้อมูลจากแบบฟอร์มจะถูกเก็บไว้เท่าที่จำเป็นต่อการดำเนินการตามวัตถุประสงค์ข้างต้น และจะถูกลบเมื่อไม่มีความจำเป็นต้องใช้แล้ว ส่วนข้อมูลใน localStorage อยู่ในความควบคุมของผู้ใช้และคงอยู่จนกว่าผู้ใช้จะลบเอง

7. สิทธิของเจ้าของข้อมูล

ภายใต้ PDPA ผู้ใช้มีสิทธิขอเข้าถึง ขอแก้ไข ขอลบ ขอคัดค้าน หรือขอระงับการใช้ข้อมูลส่วนบุคคลของตนที่ได้ให้ไว้ผ่านแบบฟอร์ม โดยติดต่อผ่านช่องทางในข้อ 11 ทั้งนี้ ครุจิตจะดำเนินการตามสมควรเท่าที่อยู่ในวิสัยที่ทำได้สำหรับเว็บไซต์อิสระ

8. ความปลอดภัยของข้อมูล

เว็บไซต์ให้บริการผ่านการเชื่อมต่อที่เข้ารหัส (HTTPS) อย่างไรก็ตาม ไม่มีการรับส่งข้อมูลผ่านอินเทอร์เน็ตใดที่ปลอดภัยอย่างสมบูรณ์ จึงขอให้ผู้ใช้ใช้ดุลพินิจในการส่งข้อมูลผ่านแบบฟอร์ม

9. ข้อมูลของผู้เยาว์

ครุจิตเป็นเว็บไซต์เนื้อหาเชิงวิชาชีพ ไม่ได้มุ่งเก็บข้อมูลส่วนบุคคลจากผู้เยาว์เป็นการเฉพาะ และขอความร่วมมือไม่ส่งข้อมูลของบุคคลที่สามหรือเด็กผ่านแบบฟอร์ม

10. การเปลี่ยนแปลงนโยบาย

ครุจิตอาจปรับปรุงนโยบายนี้เป็นครั้งคราวเพื่อให้สอดคล้องกับการใช้งานจริงหรือข้อกำหนดทางกฎหมาย โดยจะระบุวันที่ปรับปรุงล่าสุดไว้ที่ส่วนต้นของหน้า

11. ติดต่อเรา

หากมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัว หรือต้องการใช้สิทธิเกี่ยวกับข้อมูลส่วนบุคคล ติดต่อผู้พัฒนาเว็บไซต์ได้ที่ burapatis@gmail.com หรือผ่าน แบบฟอร์มแจ้งข้อมูล/เสนอแนะ
